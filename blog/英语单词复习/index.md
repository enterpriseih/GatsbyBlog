# Word

## dispose

### Pronunciation

dispose[dɪ'spəʊz]

```audio
name: dispose
isSound: true
```

### Meaning

v. 使有益于，使倾向于，排列，布置，丢掉，处理

### Root

1. 前缀：dis，分开，朝不同的方向
2. 词根：pos，放置，安置
3. 记忆：dis 分开 + pos 放置+ e = dispose 处置

### Phrase

1. dispose of：丢掉，清除，处理

### Derivative

1. dispos`al`：清除，处理
2. dispos`able`：一次性的，可自由支配的

### Sentence

```audio
name: Her sense of humor disposed me to like her.
isSound: true
```

## authorize

### Pronunciation

authorize['ɔ:θəraɪz]

```audio
name: authorize
isSound: true
```

### Meaning

v. 批准，准许

### Root

1. 词根：auth 作者
2. 后缀：or 名词后缀，与 ... 相关的人（某地人，职业）
3. 后缀：ize 动词后缀（同 ise），表示 “...化”，照 ... 的样子做，按 ... 的方式处理，使成为 ...

### Derivative

1. author`ise`：批准，授权
2. authorit`arian`：专制的，独裁的
3. author`ity`：官方，当权；职权；官方许可；权威人士；权威

### Sentence

```audio
name: Only the president can authorize a nuclear attack.
isSound: true
```
