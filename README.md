## 使用

```bash
// 注册 algolia 账号密码，取得相关密钥，填入 .env 文件
mv .env.example .env
yarn
yarn start // 启动本地开发服务器
yarn build // 生成构建文件
yarn serve // 根据生成的构建文件启用本地服务器
yarn deploy:coding // 部署到 coding
yarn deploy:github // 部署到 github
```

> 可能需要翻墙，否则 gatsby-plugin-sharp 安装失败。
